Project Assets folder

  1. Download the zip file by clicking the image below.
  2. Extract the zip file and rename the folder with your Project Title without spaces For example if you project tile is Create Game, your folder should be CreateGame.
  3. Replace the existing files with files from your project.
  4. Add your assets (images, pdfs, etc.) to the folder.
  5. Add the folder to _projects/project_assests folder in GitHub. Simply go to tuftsceeo github, and open projects_assests folder inside _projects folder and click and drag the folder to the GitHub WebUI.
